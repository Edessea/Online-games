# 🧙‍♂️🚲 Ultimate Trivia Challenge: Harry Potter vs Stranger Things 🪄🔦

Welcome to the **Ultimate Trivia Challenge**! Test your knowledge across two of the most iconic universes: the magical world of *Harry Potter* and the upside-down realm of *Stranger Things*.

This is a terminal-based Python game that challenges you with a random selection of questions from both universes, complete with fun ASCII art and emojis!

## ✨ Features
- **Two Universes Collide:** Questions are pulled from both *Harry Potter* and *Stranger Things*.
- **Three Difficulties:** Choose between Easy 🟢, Medium 🟡, and Hard 🔴.
- **Randomized Runs:** 10 questions are randomly selected each playthrough, meaning no two games are exactly alike!
- **ASCII Art Rewards:** Get a correct answer and you might be treated to some cool terminal ASCII art! 🎨
- **Score Tracking:** See how you hold up at the end. Are you a Trivia Master or do you need to study more?

## 🚀 How to Run

### Prerequisites
Make sure you have Python 3 installed on your system. You can check this by running `python3 --version` or `python --version` in your terminal. No external libraries are necessary!

### Running the Game
1. Open your terminal and navigate to the directory where the game files are located.
2. Run the main trivia file using Python:
   ```bash
   python3 trivia.py
   ```
   *(Note: Depending on your system setup, you might just need to use `python trivia.py`)*

3. Follow the on-screen prompts to select your difficulty and start answering!

## 🎮 How to Play
- Once the game starts, you'll be prompted to pick a difficulty level (1, 2, or 3).
- For each question, you'll be given 4 options (A, B, C, D).
- Type the letter corresponding to your answer and press `Enter`.
- Have fun and aim for that perfect 10/10 score! 🏆
